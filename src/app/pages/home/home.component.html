Open Source

WhatsApp Web API para .NET

Wap é uma biblioteca open source que conecta diretamente ao WhatsApp Web via WebSocket, implementando o protocolo completo: Noise handshake, Signal Protocol, criptografia end-to-end e mensagens de todos os tipos.

Documentação NuGet Package

Quick Start

Comece em 3 passos

dotnet add package Wap 
var client = await WapClient.CreateAsync(authFolder: "./auth");

client.OnMessage(msg =>
    Console.WriteLine(msg.TextContent));

await client.ConnectAsync();
await client.SendTextAsync(jid, "Hello from Wap!");
