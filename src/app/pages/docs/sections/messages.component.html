Mensagens

Envie e receba todos os tipos de mensagem pelo WhatsApp.

Enviar Texto

Envie uma mensagem de texto simples para um contato ou grupo.

Reações

Envie reações (emojis) para mensagens existentes.

Localização

Envie uma localização com coordenadas e nome do local.

Contato

Envie um cartão de contato no formato vCard.

Receber Mensagens

Escute o evento OnMessage para receber todas as mensagens em tempo real.

Confirmação de Leitura

Envie confirmação de leitura (check azul) para mensagens recebidas.
