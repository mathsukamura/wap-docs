Conexão

Como conectar ao WhatsApp Web, lidar com QR Code e gerenciar a conexão.

Conexão Básica

Crie o client informando a pasta de autenticação e conecte. Na v2.0.0, o QR Code é buscado automaticamente do servidor e exibido no terminal no primeiro acesso. As credenciais são salvas automaticamente após login e upload de pre-keys.

Eventos de Conexão

Escute os eventos de conexão para saber quando o client conectou, receber o QR Code ou tratar desconexões.

Desconectar

Para encerrar a conexão de forma limpa, utilize o método DisconnectAsync.
