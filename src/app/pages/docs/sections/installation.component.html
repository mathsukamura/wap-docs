Instalação

Como instalar e configurar o Wap no seu projeto .NET.

Pré-requisitos

  • 1 .NET 8 ou superior instalado
  • 2 NuGet packages são instalados automaticamente
1

Instalar o pacote

Adicione o pacote Wap ao seu projeto via NuGet.

2

Importar os namespaces

Importe os namespaces necessários no seu arquivo C#.

3

Criar e conectar o client

Crie uma instância do WapClient e conecte ao WhatsApp Web.
