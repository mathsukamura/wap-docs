Conheça o Wap, uma biblioteca C# para a API do WhatsApp Web.

O que é o Wap?

Wap é uma biblioteca C# completa para interagir com a API do WhatsApp Web, criada por mathsukamura. Ela implementa o protocolo Signal para criptografia de ponta a ponta com multi-device fanout, handshake Noise_XX com protobuf, autenticação automática via QR Code e Pairing Code, envio e recebimento de todos os tipos de mensagem, upload e download de mídia com criptografia, gerenciamento de grupos com Sender Key distribution, comunidades, newsletters e muito mais.

Principais Recursos

  • Signal Protocol E2E com multi-device fanout
  • Noise_XX handshake com protobuf ClientPayload
  • QR Code automático / Pairing Code auth
  • Send / receive all message types
  • Media upload / download with encryption
  • Groups com Sender Key distribution / communities / newsletters
  • App State Sync
  • Auto-reconnect
